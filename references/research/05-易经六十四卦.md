## 一、易经六十四卦概述

### 1. 易经的本质
> 来源：天纪-人间道.txt 第6页

```
易经本身並不難，故讀者先要有一概念，在讀易之前，平心静氣，捨去一切雜
念，千萬不可在讀易之前，先對易經有所認定，那尚未開始，您已經錯了。
```

### 2. 人间道的定义
> 来源：天纪-人间道.txt 第7页

```
易經中共分六十四卦，每一卦體代表事，代表一狀況每一卦有六爻，每個爻意
味事之時機也。每一卦的爻是由下而上，從第一爻到第六爻，共六階段，一般只知
其為事之演進，吾人在何階段，做何動作，都可以預知其未來之果。
```

### 3. 六爻定位
> 来源：天纪-人间道.txt 第7页

```
吾人先定位，六爻可區分為六個層面：
- 初爻：基层/起始阶段
- 二爻：中层/发展阶段
- 三爻：下卦之顶/危险阶段
- 四爻：近君位/谨慎阶段
- 五爻：君位/成功阶段
- 上爻：极点/过度阶段
```

### 4. 学习方法
> 来源：天纪-人间道.txt 第6页

```
吾人的邏輯〔求學的方法〕如下：假設→驗證→結果，凡天下任何事都無法離
開這個科學精神，是非應辨，真理即現。
```

## 二、重点卦位详解

### 1. 乾为天 ䷀（第11-15页）
> 来源：天纪-人间道.txt 第11页

**卦象**：六龍御天之課 廣大包容之象

**卦图象解**：
- 一、官人在梯上：平步青雲象
- 二、鹿在雲中：作為行事心中不可求祿，則祿至矣
- 三、石上玉有光明：示人才如玉之含光，吉也
- 四、人磨玉：示人堅心不移，夕惕若厲，隨時戒之
- 五、工匠：有重立門户之象
- 六、梯上官人：棺也。卜疾厄凶也
- 七、官人望磨玉人：示人即平步青雲須戒盛，且望下看，念中無妄，則升遷平安

**乾：元、亨、利、貞。**
> 来源：天纪-人间道.txt 第11页

```
元為萬物之始生，亨為萬物之成長，利為萬物之遂成，貞為萬物之堅心，此為乾之性，恒
古不變，示人须有堅定不移之心志，萬事乃成於此。
```

**六爻详解**：

| 爻位 | 爻辞 | 人事应用 |
|------|------|----------|
| 初九 | 潛龍勿用 | 韬光养晦，等待时机。如舜之田漁，其德已著 |
| 九二 | 見龍在田，利見大人 | 德已显现，利见大德之人。謹言慎行，但求處於無過之地 |
| 九三 | 君子終日乾乾，夕惕若厲，无咎 | 朝夕不懈，随时警惕。進德脩業，知進退之機 |
| 九四 | 或躍在淵，无咎 | 进退须待时。時行時止，不可恆久不變 |
| 九五 | 飛龍在天，利見大人 | 进乎天位君位。廣納天下大德之人共成之 |
| 上九 | 亢龍有悔 | 位之極點，過此時太過也。知進退存亡而終至無過 |

**用九：見群龍無首，吉**
> 来源：天纪-人间道.txt 第12页

```
用九之道，即用鲍陽剛之道，須剛柔相濟方為中道，如過乎剛，以至剛為天下之優先，此
凶之道也。
```

**文言四德**：
> 来源：天纪-人间道.txt 第12页

```
元者，善之長也；亨者，嘉之會也；利者，義之和也；貞者，事之幹也。
君子體仁，足以長人。嘉會，足以合禮。利物，足以合義。貞固，足以幹事。
```

**大人之德**：
> 来源：天纪-人间道.txt 第15页

```
夫大人者，與天地合其德，與日月合其明，與四時合其序，與鬼神
合其吉凶，先天而天弗爲，後天而奉天時，天且弗違，而況於人乎？
```

### 2. 坤为地 ䷁（第17-19页）
> 来源：天纪-人间道.txt 第17页

**卦象**：生載萬物之課 君倡臣和之象

**卦图象解**：
- 一、十一口：吉也，陳姓人也
- 二、一官人：倌人，丈夫也。公務員也
- 三、坐看二串錢：欲拿不得，憂心忡忡狀
- 四、一馬者：肖馬人；指官貴；指調動也
- 五、金甲神：示民心也
- 六、授文書與官：授官封侯
- 七、土堆上有四點：黑也，背有陰謀也

**坤：元、亨、利牝馬之貞。**
> 来源：天纪-人间道.txt 第17页

```
坤乃地之厚德，示人效地之性，如牝馬之柔順而健行也，貞示堅心，但與乾不同，乃堅心
於柔順之念也。
```

**六爻详解**：

| 爻位 | 爻辞 | 人事应用 |
|------|------|----------|
| 初六 | 履霜，堅冰至 | 戒小人於初。小人始雖甚微，但絶不可使其長 |
| 六二 | 直方大，不習，无不利 | 直者，直來直往，不迂迴作假。方者，原則不變。大者，容載萬物 |
| 六三 | 含章可貞，或從王事，无成有終 | 不居其功，將善歸於君王。為臣之道 |
| 六四 | 括囊，无咎，无譽 | 藏口而不露，則平安無災。沈默之功 |
| 六五 | 黄裳，元吉 | 臣居尊位，须有黄裳之戒，應守中而居下 |
| 上六 | 龍戰於野，其血玄黃 | 陰至極則陽至，故有抗爭。道因陰至極而無道 |

**用六：利永貞**
> 来源：天纪-人间道.txt 第19页

```
此為用陰之道。陰道柔而難常，有朝令夕改之憂。故利在常而堅固。
```

**坤道核心**：
> 来源：天纪-人间道.txt 第17页

```
先迷後得，主利。陰從陽也，即不了解但知須從陽剛之中正，如不知而欲進，則迷錯，
損失在己，須居於後，則利。
```

### 3. 地天泰 ䷊（第39-40页）
> 来源：天纪-人间道.txt 第39页

**卦象**：天地交泰之課 小往大來之象

**卦图象解**：
- 一、月中桂開：清明政治之時
- 二、官人登梯：升遷順遂之意，官人亦倌人，即丈夫也
- 三、鹿銜書：受天子恩賜祿位
- 四、小兒在雲中：少年得志象。得天官貴人助也
- 五、羊回頭：回陽間也。亦楊姓人，發肖羊人成格

**泰，小往大來，吉亨。**
> 来源：天纪-人间道.txt 第39页

```
陽氣下降，陰氣上升，陰陽和暢，萬物生焉，此天地之泰。人間之泰，大則居上，小則臣
下，君王推誠任下，臣盡誠以事君，上下同志，朝廷之泰。君子，小人亦如是，君子處君位，
小人居下位，天下之泰也。
```

**六爻详解**：

| 爻位 | 爻辞 | 人事应用 |
|------|------|----------|
| 初九 | 拔茅茹，以其彙，征吉 | 同類相聚，如拔茅之根相牽連，同欲上進 |
| 九二 | 包荒，用馮河，不遐遺，朋亡，得尚于中行 | 治泰之道，主將位而能包荒，須稟持公正之態度 |
| 九三 | 无平不陂，无往不復，艱貞无咎 | 居安思危，泰久而必否，須堅貞可常保泰 |
| 六四 | 翩翩不富，以其鄰，不戒以孚 | 誠意相合，不待告誡而誠意相待 |
| 六五 | 帝乙歸妹，以祉元吉 | 任用剛中之賢，志同相交 |
| 上六 | 城復于隍，勿用師，自邑告命，貞吝 | 泰之終也，小人處之，行必否矣 |

**关键启示**：
> 来源：天纪-人间道.txt 第39页

```
內陽而外陰，內健而外順，內君子而外小人，君子道長，小人道消也。
```

### 4. 天地否 ䷋（第41-42页）
> 来源：天纪-人间道.txt 第41页

**卦象**：天地不交之課 人口不實之象

**卦图象解**：
- 一、男子臥病：男子為陽，陽同佯，裝病之象也
- 二、镜破：不可赴也。破鏡難圓之象
- 三、人路上坐：小人阻路，不讓進也
- 四、張弓箭頭落地：鳥盡弓藏也
- 五、人拍掌笑：幸災樂禍之小人，做假攻訐兩人感情得逞狀，夫妻凶
- 六、舌：主口舌，官司，謡言生害
- 七、口上有四點：四點為黑，暗中誣陷之謡言，古人謂三口成虎

**否之匪人。不利君子貞，大往小來。**
> 来源：天纪-人间道.txt 第41页

```
舉凡天地之間皆為人道，今天地不交，不生萬物，是無人道，故曰匪人。處此之時，君子
不可堅心，因正道不行，小人道長之時。
```

**六爻详解**：

| 爻位 | 爻辞 | 人事应用 |
|------|------|----------|
| 初六 | 拔茅如，以其彙，貞吉，亨 | 君子處否時而居下位，冀得同類而進，堅守其節 |
| 六二 | 包承，小人吉，大人否，亨 | 大人處否，以道之陽剛自勉，絶不枉屈正道 |
| 六三 | 包羞 | 不中不正之人居否之時，極盡小人之能事 |
| 九四 | 有命，无咎，疇離祉 | 凡事皆由君命而出，則無災，且大志得行 |
| 九五 | 休否，大人吉，其亡，其亡，繋于苞桑 | 惟有陽剛中正之人居君位，能去天下之否 |
| 上九 | 傾否，先否後喜 | 否極泰來 |

**关键启示**：
> 来源：天纪-人间道.txt 第41页

```
內陰而外陽，內柔而外剛，內小人而外君子，小人道長，君子道消也。
```

**处否之道**：
> 来源：天纪-人间道.txt 第41页

```
君子以儉德辟難，不可榮以祿。處否之時，損儉自己，避開禍亂。千萬不可榮居祿位，
戀棧不去。因小人得志之時，君子仍居顯位，則禍必及己身。
```

### 5. 水雷屯 ䷂（第11页）
> 来源：天纪-人间道.txt 第11页

**卦象**：物之初始，如胎在母体中

**人事应用**：
```
屯為物之初始，如胎在母體中，故萬物之始生謂屯，此卦示吾人事之始生，如何戒之於初，
即「慎始」之精神。
```

**六爻定位应用**：
- 例如，你是一位新任銀行經理，剛到任時，即物之始生
- 如你剛考上金融特考而分發到一陌生之單位，那你就是居於第一爻的位置
- 如你是大企業的第二代須接君位，你就必須看第五爻君位處屯時之方法

**关键启示**：
```
唐朝名將李靖，有一銘言：「古今勝敗，一誤而已，」示人一念之間，就已決定勝敗，
故開始時是最重要的。
```

## 三、其他卦位应用（部分摘录）

### 12. 天火同人 ䷌（第33-34页）
> 来源：天纪-人间道.txt 第33页

**卦象**：游魚從水之課 二人分金之象

**卦图象解**：
- 一、人捧文書上有心字：得民心之象，同心協力之象。寧姓
- 二、人張弓射山上：高中金榜。張姓人氏
- 三、一鹿飲水：财祿滾滾而來。亦在野之賢人
- 四、一溪：平和安靜狀
- 五、鹿之性，動則敏，靜則順，此剛健人之性

**同人：于野，亨。利涉大川，利君子貞。**
> 来源：天纪-人间道.txt 第33页

```
野即遠與外之意，以天下大同之道，聖賢大公之心，使之無遠不同。常人之同，以私意所
合，此暱親之情。故必于野，方可謂大同。
```

**人事应用**：
- 官家：必主人和，手下得力，禄命九重来
- 商人：财禄丰盛，但须二人合伙方可成事
- 自由业：必主名气大，声名远播
- 对妻不利：妻必成明夷卦，主妻有暗伤

**关键启示**：
> 来源：天纪-人间道.txt 第33页

```
文明以健，中正而應，君子正也。唯君子爲能通天下之志。
```

### 13. 火天大有 ䷍（第35-36页）
> 来源：天纪-人间道.txt 第35页

**卦象**：金玉滿堂之課 日麗中天之象

**卦图象解**：
- 一、婦人腹中一道氣：妊娠之事；事萌於內也
- 二、氣中二子：主雙喜臨鬥。伯仲之間象
- 三、一藥王：平安之象。得病有救，遇良醫也
- 四、藥有光：藥有名也，災中有救也
- 五、女人受藥：女人為陰，示表面不信，但暗中仍聽之
- 六、一犬：乃狗年，戌月，戌日，此論時機也；或狄姓人士

**大有：元亨。**
> 来源：天纪-人间道.txt 第35页

```
此乃以卦才而言，不言元亨利貞四德，乃恐與乾坤二卦相混淆，只言元亨乃盡元之義，元
乃物之始，萬物生成之初皆為大善，因必有功用。
```

**人事应用**：
- 未婚之女：娇美如花，事业功成名就，但婚姻不好
- 性阳刚过人，倔强固执
- 不重视婚姻，唯对子女独有所钟
- 肖犬之小姐：必主大吉大利

**关键启示**：
> 来源：天纪-人间道.txt 第35页

```
君子以遏惡揚善，順天休命。火因在天上，故能照見萬物，君子觀大有之象，故知治眾之道，
唯遏惡揚善，此能順天命而安眾民心。
```

### 14. 地山谦 ䷎（第37-38页）
> 来源：天纪-人间道.txt 第37页

**卦象**：地中有山之課 仰高就下之象

**卦图象解**：
- 一、月當天：清明之政，中秋時也
- 二、一人骑鹿：才祿俱備
- 三、三人腳下亂絲，隱於山後：小人不敢正面示人，惟隱山謙之後實則亂如麻
- 四、貴人捧鏡：明鏡高懸，執法公正光大
- 五、文字上有公字：處事以公，得理也

**謙：亨，君子有終。**
> 来源：天纪-人间道.txt 第37页

```
有德而不居，為謙，人能以謙遜自處，無往不利。君子之志在達理，樂天命而無競之心，
但求內充，退讓而不衿持。能安於謙，終身不易。
```

**人事应用**：
- 贤妻良母：任劳任怨，谦以让人
- 夫卦为遯：主夫妻不利，感情退化
- 身体：小病痛不断，因劳成疾多有

**关键启示**：
> 来源：天纪-人间道.txt 第37页

```
天道虧盈而益謙。地道變盈而流謙。鬼神害盈而福謙。人道惡盈而好謙。
```

### 15. 雷地豫 ䷏（第39-40页）
> 来源：天纪-人间道.txt 第39页

**卦象**：鳳凰生雛之課 萬物發生之象

**卦图象解**：
- 一、兩重山：出象，千里阻隔也
- 二、官人在中：外出不任官象；丈夫外出從商
- 三、一祿一馬：從商之人，財祿豐盛
- 四、金銀數錠：紙錢錠，主喪服
- 五、錢一堆者：內有憂心忡忡象

**豫：利建侯，行師。**
> 来源：天纪-人间道.txt 第39页

```
聖人體豫悦之道，知兵師之興，諸侯之封，必眾心和悦方成，故能君臨萬邦，能群聚大眾，
唯和悦可也，此豫之時也。
```

**人事应用**：
- 因禄远行之象，大利肖马之人
- 事业：财官双美，平步青云
- 婚姻：延后，比同时出生之人晚五年婚
- 婚事：有娶二婚妻之象，或年长于己之小姐

**关键启示**：
> 来源：天纪-人间道.txt 第39页

```
天地以順動，故日月不爲過，而四時不忒；聖人以順動，則刑罰清，而民服。
```

### 16. 泽雷随 ䷐（第41-42页）
> 来源：天纪-人间道.txt 第41页

**卦象**：良士琢玉之課 如水推車之象

**卦图象解**：
- 一、雲中雁傳書：飛來喜訊象
- 二、一堆錢：財聚也
- 三、朱門內有人坐：君王之象，必見君面也
- 四、一人在鬥外立地：乃士人求仕，受尊重之象
- 五、串錢在地：有用利不行之象
- 六、亂珠聚盆：先聚後散之象

**隨：元，亨，利，貞，无咎。**
> 来源：天纪-人间道.txt 第41页

```
隨之道，可以致大亨通。君子之道，眾人所隨，又遇事擇所隨，隨能得道，必致大亨。今
人君之從民意，臣下之從君意，人人隨從義，利在堅守隨義，隨因能正，其道必亨。
```

**人事应用**：
- 女同性恋倾向：心念一直认为自己为男人
- 事业：全心于事业，为达目的不择手段
- 化解法：用巽为风卦可解此局

**关键启示**：
> 来源：天纪-人间道.txt 第41页

```
隨時之義大矣哉。君子之道，隨時而變，因時制宜。
```

### 17. 山风蛊 ䷑（第43-44页）
> 来源：天纪-人间道.txt 第43页

**卦象**：三蠱食血之課 以惡害義之象

**卦图象解**：
- 一、孩兒在雲中：有天官促成也。不求名利之人
- 二、雁啣書：有消息至
- 三、一鹿：有大利也
- 四、錢在空中：此心中圖利之象也
- 五、男女相拜：男女相合，此為利而結合，非正婚也

**蠱：元亨，利涉大川。**
> 来源：天纪-人间道.txt 第43页

```
蠱之大者，乃時之艱難險阻，能有治蠱之才，則必亨通。
```

**先甲三日，後甲三日**
> 来源：天纪-人间道.txt 第43页

```
甲為十干之首，意言事之始也。先甲三日，乃言治蠱之道，應先思其事之原，再思其事之
後，乃知救弊之道。亦強調慮之深，推之遠也。
```

**人事应用**：
- 男人图利野心至大，为达目的不择手段
- 科甲：读书考试成绩必超人一等
- 婚期：比同时生人提前四年婚

**关键启示**：
> 来源：天纪-人间道.txt 第43页

```
有始必有終，終之後必有始，此天道也。聖人體始終之道，故能於事之始而探究其所以然，
能於事之終而備其將然，此先甲後甲之慮，故能治蠱而天下亨。
```

### 18. 地泽临 ䷒（第45-46页）
> 来源：天纪-人间道.txt 第45页

**卦象**：鳳入雞群之課 以上臨下之象

**卦图象解**：
- 一、婦人乘風：外援為陰女，美人也
- 二、一車上有使旗：出使之象，以謀略可解災
- 三、人在山頂頭：盛極將衰，招困也，即將下坡也
- 四、虎在山下坐：人受虎困之象
- 五、一合：先成後破象
- 六、人射弓：張姓人士，夷狄之人，以弓射獵，故外族入侵象

**臨：元，亨，利，貞。至于八月有凶。**
> 来源：天纪-人间道.txt 第45页

```
言臨之道，可致大亨也。二陽才剛生於下，聖人戒之於方盛，方盛而慮衰，則可以防自滿，
圖其永久，衰而後戒，無及也，此乃戒盛之道。
```

**人事应用**：
- 已婚妇人：夫宫必无力
- 夫业受困：因妻之过刚
- 子女：母亲发展权限过大，终致子女反弹

**关键启示**：
> 来源：天纪-人间道.txt 第45页

```
剛浸而長，説而順，剛中而應，大亨以正，天之道也。
```

### 19. 风地观 ䷓（第47-48页）
> 来源：天纪-人间道.txt 第47页

**卦象**：雲捲晴空之課 春花競發之象

**卦图象解**：
- 一、日月當天：政治清明，普照大地，父母雙全
- 二、官人香案邊立：倌人，丈夫也。蔡姓人士，神人共明之象
- 三、鹿在山上：得高祿之象
- 四、金甲人：君側之人也，護衛也
- 五、執印秤：提印信而來授權也

**觀：盥而不薦，有孚顒若。**
> 来源：天纪-人间道.txt 第47页

```
人於祭祀之始，求神之時，心念最專精之時也。居上位者，正其儀表，以為下民之觀，當
始與終，皆能令民之觀如初始之專精，則盡觀之道也。
```

**人事应用**：
- 长女卦：婚後局消
- 以财富论定：是否愿意嫁人
- 嫁二婚夫：且对方有子之人

**关键启示**：
> 来源：天纪-人间道.txt 第47页

```
大觀在上，順而巽，中正以觀天下。觀天之神道，而四時不忒，聖人以神道設敎，而天下服矣。
```

### 20. 火雷噬嗑 ䷔（第49-50页）
> 来源：天纪-人间道.txt 第49页

**卦象**：日中爲喜之課 順中有物之象

**卦图象解**：
- 一、北斗星：君王也，夫君也，公職也
- 二、婦人燒香拜：妻於人事無助，乃求助於天
- 三、憂字不全喜字全：乃無又喜象
- 四、一雞食稻：果被食，無果也
- 五、一錢財、一鹿者：於財祿事憂心忡忡也
- 六、禽：字解為『離多會少』之象

**噬嗑：亨，利用獄。**
> 来源：天纪-人间道.txt 第49页

```
此即天下之事不能亨通，乃因有間阻，非刑獄何以除之。
```

**人事应用**：
- 女性刚：报复手段常用咬的，平日唠叨不停
- 婚期：延后六年
- 事业：非常顺利

**关键启示**：
> 来源：天纪-人间道.txt 第49页

```
剛柔分，動而明，雷電合而章。照與威並行，為用獄之道。人能照，則無隱情，有威則無人不畏。
```

## 四、易经决策方法论

### 1. 时位决策法
> 来源：天纪-人间道.txt 第7页

```
吾人在何階段，做何動作，都可以預知其未來之果。如更進一步的推論，則是您在處於
何階段，應如何做，如果做錯結果會如何，易經上早已明示。
```

**应用原则**：
- 初爻阶段：韬光养晦，积蓄力量
- 二爻阶段：崭露头角，寻求贵人
- 三爻阶段：谨慎行事，进退有度
- 四爻阶段：等待时机，伺机而动
- 五爻阶段：大展宏图，广纳贤才
- 上爻阶段：知进退，防过极

### 2. 以果决行法
> 来源：天纪-人间道.txt 第7页

```
易經上早已明示，更進一步提示讀者，這是一部終極的書，它可涵蓋一切，千萬不可被
自已對易的觀念鎖定，要爭脱易的束縛，吸收為己用。
```

**应用原则**：
- 先推算结果（果），再决定行动（行）
- 不解释原因，只告知结果和应对
- 命理是工具，不是玄学讨论

### 3. 刚柔相济法
> 来源：天纪-人间道.txt 第12页

```
用九之道，即用鲍陽剛之道，須剛柔相濟方為中道，如過乎剛，以至剛為天下之優先，
此凶之道也。
```

**应用原则**：
- 阳刚之道须刚柔相济
- 过刚则凶，过柔则弱
- 中道为上
