> 综合徐子平、万民英、沈孝瞻、任铁樵、韦千里、袁树珊七位八字命理大师的核心方法论，融合格局派、旺衰派、调候派三大流派精华，构建统一、可操作的八字命理分析框架。

## 使用时机

当用户请求以下类型的帮助时启用：
1. **八字排盘与分析**：请求排盘、命局解读

## 框架概览

本框架以八字命理史为纵轴，融合七位大师的方法论精华，不以某一家为尊，而是提供一张可对照、可选择的**方法论地图**。当面对一个八字时，不是机械套用某派理论，而是从多个角度交叉验证。


## 第一维度：格局论命（以沈孝瞻为核心，万民英为参考）

### 沈孝瞻格局论命五步法

**第一步：看月令取格**
「八字用神，专求月令」——以日干配月令地支，看生克关系而定格。

**第二步：辨顺用逆用**
| 类型 | 格局 | 处理方法 |
|------|------|---------|
| 善用神（顺用） | 财、官、印、食 | 保护扶助，顺势生发 |
| 不善用神（逆用） | 杀、伤、刃、劫 | 制伏控制，使其为我所用 |

**第三步：判成败救应**
- 成格：用神得到完整配合（有相神辅助），无刑冲破害
- 破格：用神被克制、被合化、被混杂破坏
- 救应：破格后有其他干支来补救（因败得成）

**第四步：审有情有力**
- 「有情」：用神相神间配合和谐
- 「有力」：用神相神有根气、透干得地

**第五步：论行运**
- 大运对格局的成格与变格影响
- 运干运支对用神喜忌的作用

### 万民英补充维度
- 神煞辅助判断（天乙贵人、华盖、羊刃等）
- 纳音参考（六十甲子纳音取象）
- 当格局判断出现分歧时，以《三命通会》为多源对照

## 第二维度：旺衰论命（以任铁樵为核心，韦千里为操作模板）

### 四维判断法（任铁樵）

| 维度 | 含义 | 权重 |
|------|------|------|
| **得令** | 日主五行与月令关系 | 最重要（得时者，虽只一干，可当半壁） |
| **得地** | 日主在地支是否有根（禄旺库余气） | 次重要 |
| **得生** | 是否有印星生扶 | 辅助判断 |
| **得助** | 是否有比劫同类相助 | 辅助判断 |

### 身强身弱判断

```
身强：得令 + (得地、得生、得助中≥2)
身弱：失令 + (得地、得生、得助中≤1)
太旺/极旺：三者全满，且无制化
太弱/极弱：三者全空，且无生扶
```

### 取用神体系

| 类型 | 适用条件 | 方法 |
|------|---------|------|
| 扶抑用神 | 正常强弱 | 身弱用印比扶身，身强用食伤财官抑身 |
| 通关用神 | 两五行相战 | 取中间五行调和（如金木战取水） |
| 调候用神 | 寒暖燥湿失衡 | 寒用火暖、热用水润、燥用癸湿、湿用丙燥 |

### 韦千里八步评断法（操作流程）

```
①看强弱 → ②定格局 → ③取用神 
→ ④论喜忌 → ⑤查岁运 → ⑥推六亲 
→ ⑦评性情 → ⑧断事业
```

这八步是现代命理判读的标准流程，韦千里的最大贡献是将模糊的"江湖诀"转化为可传授、可复制的操作程序。

## 第三维度：调候论命（以《穷通宝鉴》为核心）

### 气候优先原则

「论命惟以月令用神为主，然亦须配气候而互参之。」——沈孝瞻原已提及，余春台、徐乐吾发扬光大。

### 寒暖燥湿四态

| 气候状态 | 特征 | 优先调候 |
|---------|------|---------|
| 寒 | 冬生、水旺、无火 | 用丙火（太阳） |
| 热 | 夏生、火旺、无水 | 用壬水（江湖） |
| 燥 | 秋生、土金旺 | 用癸水（雨露） |
| 湿 | 春生、木水旺 | 用丙丁火（温暖） |

### 十天干实物说（《穷通宝鉴》特色）

- 甲木（栋梁）→ 需庚金斧斫方可成材
- 乙木（花草）→ 需癸丙滋润温暖
- 丙火（太阳）→ 需壬水映衬方光彩焕发
- 丁火（灯烛）→ 需甲木为燃料
- 戊土（城墙）→ 需甲木疏松、丙火照耀
- 己土（田园）→ 需丙火温暖、癸水滋润
- 庚金（顽金）→ 需丁火炉火锻炼方可用世
- 辛金（珠宝）→ 需壬水淘洗方显其贵
- 壬水（江湖）→ 需戊土堤防方不泛滥
- 癸水（雨露）→ 需庚金为源头、乙木为依托

## 四大流派综合解盘流程

### 步骤一：命局总览
排出八字四柱，标注干支五行、藏干、十神。

### 步骤二：格局判断（格局派视角）
1. 看月令是什么十神（正官/七杀/财/印/食神等）
2. 是否有透干会支？定格
3. 格局是否成立？有无破格？有无救应？
4. 结论：此命贵气层次的大致范围（非实指具体职位）

### 步骤三：日主评估（旺衰派视角）
1. 日主在月令旺相休囚？
2. 是否有根？几个根？
3. 生扶力量（印星）还是克泄力量（官杀伤财）占优？
4. 结论：身强/身弱/太旺/太弱

### 步骤四：调候校验（调候派视角）
1. 命局气候偏寒偏热？
2. 是否有调候用神？透干？有力？
3. 调候用神与格局用神是否冲突？
4. 结论：如冲突，以调候为急

### 步骤五：用神统合（三大派交叉验证）
| 用神维度 | 格局派答案 | 旺衰派答案 | 调候派答案 |
|---------|----------|----------|----------|
| 用神是什么 | 月令定格之物 | 平衡日主之物 | 调节气候之物 |
| 优先度 | 格局优先 | 日主优先 | 气候优先 |

**统合原则**：
1. 三派答案一致 → 大概率正确
2. 两派一致 → 以一致方为主，不一致方为辅助
3. 三派全不一致 → 信息不足，标注不确定性，列出三种可能性

### 步骤六：大运流年（袁树珊考订法 + 韦千里分运法）
1. 排大运（袁树珊考订的起运法）
2. 大运对格局的成格/变格影响（沈孝瞻运限法）
3. 流年与用神喜忌的关系（韦千里流年法）
4. 十年大运上下各五年分看（韦千里特色）

### 步骤七：六亲事业评断（韦千里八步法）
看强弱→定格局→取用神→论喜忌→查岁运→推六亲→评性情→断事业

## 流派分歧速查表

当不同流派对同一八字给出不同判断时，此表帮助定位分歧来源：

| 分歧类型 | 格局派认为 | 旺衰派认为 | 融合方略 |
|---------|----------|----------|---------|
| 财官旺但身弱 | 格局成立，贵 | 身弱不胜财官，贫 | 格局定框架→大运扶身时才能担财官 |
| 身强但格局不纯 | 格局破败 | 日主健旺，好运能成 | 格局限制上限，旺衰决定下限 |
| 调候与格局冲突 | 格局优先 | 调候优先 | 格局看先天禀赋，调候看运势时机 |

## 诚实边界

### 本框架能做到的
- 整合七位大师的核心方法论，提供多维度分析
- 清晰对比三大流派（格局、旺衰、调候）的差异
- 提供可操作的八步解盘流程
- 引用具体古籍原文作为分析依据

### 本框架做不到的
1. **不能替代算命师傅的经验**：命理需要大量实战积累，框架提供的是方法论而非经验直觉
2. **不能保证100%准确**：八字命理本身不是精密科学，不同流派对同一八字的解读可能不同
3. **不能预测具体事件**：只说趋势倾向，不说具体日期和事件
4. **不能处理极端特殊命局**：特殊格局（如从格、化气格等）需要单独处理
5. **不能替代专业命理执业者**：本框架为学习参考和知识整合之用
